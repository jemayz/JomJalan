<h1 align="center">JomJalan ‚Äì AI Smart Travel App</h1>


**JomJalan** is an intelligent, all-in-one travel app designed to solve information overload for travelers in Malaysia. Instead of switching between Google, TikTok, and Instagram, JomJalan scrapes real-time social trends to find the hottest spots and then uses AI to build a personalized itinerary based on user **budget** and **interests**.


---

## Key Features

- **Live Social Scraping**  
  Scrapes travel blogs (like *KL Foodie*) to discover trending spots, simulating the "pulse" of social media.

- **AI Travel Assistant**  
  A chatbot powered by the **Google Gemini API** that builds a custom travel plan when you provide a destination, budget, and interests.

- **Smart Map Search**  
  Fully integrated **Google Maps interface** with a custom dark-mode theme, allowing users to search for any location.

- **Itinerary Builder**  
  Add or remove scraped spots to a personal itinerary ‚Äî saved locally on the device.

- **Gamification**  
  A profile page that tracks user points and challenges (e.g., ‚ÄúVisit 3 cafes‚Äù) to increase engagement.
